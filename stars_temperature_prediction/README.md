# Прогнозирование температуры звезды


## Цель работы
Разработать нейронную сеть, которая поможет предсказывать абсолютную температуру на поверхности звезды.

## Описание данных
База обсерватории с инофрмацией о 240 звездах.

## Используемые для решения задачи модели
* Нейронная сеть

## Метрика для оценки качества моделей
RMSE

## Результат
По соотношению метрики и количества эпох лучше всего себя показала модель с нормализацией батчей. C RMSE = 4159 на 6000 эпох.

## Используемый стек
Jupyter Notebook Python: pandas, numpy, matplotlib, seaborn, phik, sweetviz, torch, sklearn.
